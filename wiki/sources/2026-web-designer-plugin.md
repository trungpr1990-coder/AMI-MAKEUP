## Summary

Plugin biến Claude thành web designer đẳng cấp thế giới bằng cách inject design thinking thực sự — typography systems, color theory, animation vocabulary, layout patterns, kỹ thuật 3D — trích xuất từ 38 website đoạt giải thưởng Awwwards 2025-2026. Giải quyết vấn đề "AI Look" (blue-purple gradients, Inter font, border-radius 12px trên mọi thứ).

## Key Points

- 48 design patterns chiến đấu-thực tế từ các site đoạt giải: card tilt 3D, torn paper dividers, giant compressed text, cursor trail effects...
- 5 palette archetypes + 5 font pairing formulas + fluid type system
- Motion design vocabulary: entrances, hovers, scroll-linked, transitions
- Anti-patterns checklist: 10 "AI Look" traps và cách phá vỡ
- **Kích hoạt on-demand** (không auto-trigger) — gọi khi cần output chất lượng designer
- File nguồn: `D:\web-designer-plugin-main\skills\web-designer\`
