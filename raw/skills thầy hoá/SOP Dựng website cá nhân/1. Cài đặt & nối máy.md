## Cách nhanh \(Windows\): chạy 1 script

Trong gói zip có sẵn `scripts/cai-dat-notebooklm-mcp.ps1`\. Mở PowerShell tại thư mục skill và chạy:

```powershell
powershell -ExecutionPolicy Bypass -File scripts/cai-dat-notebooklm-mcp.ps1
```

Script tự làm cả 3 việc: cài gói → mở Chrome cho bạn đăng nhập → đăng ký vào Claude\. **Xong thì RESTART Claude\.**

## Cách làm tay \(hiểu từng bước / cho macOS–Linux\)

### Bước 1 — Cài gói

```bash
python -m pip install --user --upgrade notebooklm-mcp-cli
```

Tìm thư mục chứa file chạy \(`nlm`, `notebooklm-mcp`\):

```bash
# Windows
python -c "import sysconfig;print(sysconfig.get_path('scripts','nt_user'))"
# macOS / Linux  (thường là ~/.local/bin)
python3 -c "import sysconfig;print(sysconfig.get_path('scripts','posix_user'))"
```

### Bước 2 — Đăng nhập Google \(trích cookie\)

```bash
nlm login
```

Mở Chrome → đăng nhập tài khoản có NotebookLM → báo **`Authentication valid!`** kèm số notebook\. Xong\.
*\(Nếu báo "không phải lệnh" → gọi bằng đường dẫn đầy đủ tới **`nlm.exe`**, hoặc thêm thư mục Scripts ở Bước 1 vào PATH\.\)*

### Bước 3 — Đăng ký MCP vào Claude

**Claude Code \(CLI\):**

```bash
claude mcp add --scope user notebooklm-mcp "<đường-dẫn-đầy-đủ>/notebooklm-mcp(.exe)"
```

**Claude Desktop:** Settings → Developer → Edit Config, thêm vào `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "notebooklm-mcp": { "command": "C:\\...\\Scripts\\notebooklm-mcp.exe", "args": [] }
  }
}
```

### Bước 4 — Restart Claude

**Bắt buộc** restart để Claude nạp công cụ\. Kiểm tra: bảo Claude *"liệt kê notebook NotebookLM"* → ra danh sách là thành công\.

## Xử lý lỗi thường gặp

- **Cài xong tool không hiện** → chưa restart Claude\. Restart lại\.

- **Báo lỗi đăng nhập / hết hạn** → chạy lại `nlm login`\. \(Việc bảo trì định kỳ duy nhất\.\)

- **`server_info`**** báo ****`stale`****/****`unverified`**** nhưng vẫn dùng được** → tự kiểm tra của công cụ hơi khó tính\. Cứ thử bảo Claude liệt kê notebook; chạy được là ổn, **không cần** login lại\.

- **Nhiều tài khoản Google** → `nlm login switch <tên-profile>` để đổi tài khoản mặc định\.

- Đừng mở 2 phiên đăng nhập song song cùng 1 tài khoản \(dễ đá cookie nhau\)\.
